### Template
```html
  <va-medium-editor>
      <h1>Some title</h1>
      <p>the text of article</p>
    </va-medium-editor>
```

`va-medium-editor` uses external lib with overridden styles. Check more information [here](https://yabwe.github.io/medium-editor/)
